# SIMADUK

SIMADUK is a village information system website built using PHP Native and MySQL.

## Features
- News management
- Public aspiration form
- Admin dashboard
- CRUD functionality

## Technologies
- PHP Native
- MySQL
- HTML
- CSS
- JavaScript
